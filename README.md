# Password Generator (Python)

Gerador de senhas seguras desenvolvido em Python, utilizando o módulo `secrets` para garantir alta segurança criptográfica.

## Funcionalidades

- Escolha da quantidade de caracteres
- Opção de incluir ou não caracteres especiais
- Geração de senha utilizando `secrets` (criptograficamente seguro)
- Senhas fortes e imprevisíveis

## Tecnologias Utilizadas

- Python 3
- Biblioteca padrão `secrets`
- Biblioteca padrão `string`
